## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

```bash
$ docker-compose up -d
```

## Running the app

```bash
# development
 npm run start

# watch mode
 npm run start:dev
```

<h1> Message management </h1>
Kafka: http://localhost:8080/ <br>
Rabbit: http://localhost:15672/
